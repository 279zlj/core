EXIF Makernotes - Reference Information

Ricoh Makernote Format Specification


There are three types of Ricoh Makernote:


Ricoh Type 1 Makernote (Text Makernote)

Field Size Description
Text Variable Text, beginning with "Rv" or "Rev"


Ricoh Type 2 Makernote (Empty Makernote)

Field Size Description
Blank Variable Blank field filled with 0x00 characters


Ricoh Type 3 Makernote

Field Size Description
Header 5 Bytes "Ricoh" or "RICOH"
Unknown 1 Byte Unknown field
Zeros 2 Bytes Two 0x00 characters
IFD Data Variable Standard IFD Data using Ricoh Tags but always uses Motorola (Big-Endian) byte alignment


For format specifications of TIFF Image File Directory data, see Section 2 of: TIFF, Revision 6.0, June 3, 1992, Adobe Systems Incorporated

Ricoh Makernote - Camera Info Sub-IFD Format

Field Size Description
Header 19 Bytes "[Ricoh Camera Info]"
Unknown 1 Byte Unknown field
IFD Data Variable NON-Standard IFD Data using Ricoh Sub-IFD Tags
Always uses Motorola (Big-Endian) byte alignment,and
there is no final Next-IFD pointer.
