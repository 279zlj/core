EXIF Makernotes - Reference Information

Konica / Minolta Makernote Format Specification


There are several types of Konica / Minolta Makernote:


Konica / Minolta Type 1 Makernote

Field Size Description
Header 3 Bytes "MLY"
Unknown Data Variable Unknown Data


Konica / Minolta Type 2 Makernote

Field Size Description
Header 2 Bytes "KC"
Unknown Data Variable Unknown Data


Konica / Minolta Type 3 Makernote

Field Size Description
Header 8 Bytes "+M+M+M+M"
Unknown Data Variable Unknown Data


Konica / Minolta Type 4 Makernote

Field Size Description
Header 5 Bytes "MINOL"
Unknown Data Variable Unknown Data


Konica / Minolta Type 5 Makernote

Field Size Description
IFD Data Variable Standard TIFF IFD with Olympus Tags

This makernote has no header - the IFD starts immediately



For format specifications of TIFF Image File Directory data, see Section 2 of: TIFF, Revision 6.0, June 3, 1992, Adobe Systems Incorporated

For details of Olympus tags, See the Olympus Makernote Page Here

All EXIF offsets are relative to the start of the TIFF header at the beginning of the EXIF segment





Minolta Camera Settings Definition (Tag 1 or Tag 3)

Camera settings are defined using binary string containing 4 byte LONG values.
The position of a value in the string determines what the value represents.
A lookup table is used to determine the meaning of each value
