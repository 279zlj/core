EXIF Makernotes - Reference Information

Pentax / Asahi Makernote Format Specification


There are two types of Pentax / Asahi Makernote:


Pentax / Asahi Type 1 Makernote

Field Size Description
IFD Data Variable NON-Standard TIFF IFD Data using Pentax Tags
IFD has no Next-IFD pointer at end of IFD, and
Offsets are relative to the start of the current IFD tag, not the TIFF header


Pentax / Asahi Type 2 Makernote

Field Size Description
Header 4 Bytes "AOC\x00"
Unknown 2 Bytes Unknown field
IFD Data Variable NON-Standard TIFF IFD Data using Casio Type 2 Tags
IFD has no Next-IFD pointer at end of IFD, and
Offsets are relative to the start of the current IFD tag, not the TIFF header


For format specifications of TIFF Image File Directory data, see Section 2 of: TIFF, Revision 6.0, June 3, 1992, Adobe Systems Incorporated
